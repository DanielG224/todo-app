# Todo App

Full-stack aplikacja do zarządzania zadaniami.

## Technologie
- React (frontend)
- Node.js + Express (backend)
- MongoDB (baza danych)

## Funkcje
- Dodawanie zadań
- Usuwanie zadań
- Oznaczanie jako wykonane

## Uruchomienie

### Backend
cd backend
node index.js

### Frontend
cd frontend
npm start
